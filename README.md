## Описание

GUI-приложение "Random Password Generator" создано с использованием библиотеки `tkinter` для генерации надежных случайных паролей. Приложение позволяет пользователю настраивать длину пароля, а также выбирать, какие типы символов (цифры, буквы, спецсимволы) будут включены в пароль.

**Основные возможности:**

*   **Настраиваемая длина пароля:** Используйте ползунок для установки желаемой длины пароля (от 8 до 64 символов).
*   **Выбор типов символов:** Включайте или исключайте цифры, буквы (в верхнем и нижнем регистре) и специальные символы.
*   **Генерация пароля:** Мгновенная генерация пароля на основе заданных параметров.
*   **Сохранение истории:** Сгенерированные пароли автоматически сохраняются в файл `password_history.json` и отображаются в истории генераций в приложении.
*   **Загрузка истории:** История генераций загружается при каждом запуске приложения.
*   **Проверка корректности ввода:** Минимальная и максимальная длина пароля установлены по умолчанию.

## Технологии

*   **Язык:** Python 3.x
*   **GUI:** Tkinter
*   **Генерация случайных чисел:** `random`
*   **Работа с символами:** `string`
*   **Сохранение данных:** JSON

## Установка и запуск

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/yourusername/your-repo-name.git
    cd your-repo-name
    ```
    (Замените `yourusername/your-repo-name.git` на URL вашего репозитория).

2.  **Убедитесь, что у вас установлен Python 3.6+**

3.  **Запустите приложение:**
    ```bash
    python password_generator.py
    ```

    (Файл `password_generator.py` должен находиться в корневой директории проекта).

## Примеры использования

1.  **Откройте приложение.**
2.  **Настройте параметры:**
    *   Передвиньте ползунок, чтобы выбрать желаемую длину пароля (например, 16 символов).
    *   Отметьте или снимите галочки у чекбоксов "Цифры", "Буквы", "Спецсимволы" в соответствии с вашими требованиями.
3.  **Нажмите кнопку "Сгенерировать пароль".**
4.  Ваш новый пароль отобразится в поле "Сгенерированный пароль".
5.  Этот пароль также будет добавлен в список "История генераций" и сохранен в файл `password_history.json`.

## Возможные тесты (по желанию)

*   **Тест 1: Генерация с разными наборами символов.**
    *   Проверьте, что при выборе только цифр генерируются пароли только из цифр.
    *   Проверьте, что при выборе всех опций генерируются пароли со смешанными символами.
*   **Тест 2: Сохранение и загрузка истории.**
    *   Сгенерируйте несколько паролей.
    *   Закройте приложение.
    *   Откройте приложение снова и убедитесь, что сгенерированные ранее пароли отображаются в истории.
    *   Проверьте содержимое файла `password_history.json`.
*   **Тест 3: Корректность длины.**
    *   Убедитесь, что ползунок не позволяет установить длину меньше 8 или больше 64.
*   **Тест 4: Обработка ошибок.**
    *   Попробуйте снять галочки со всех типов символов и нажать "Сгенерировать пароль". Должно появиться предупреждение.
